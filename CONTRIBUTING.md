# Contributing to Brand Asset Generator

Contributions are welcome! Whether it's a bug fix, new engine, or documentation improvement.

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/new-engine`)
3. Make your changes
4. Run tests and lint
5. Submit a Pull Request

## Development

```bash
git clone https://github.com/BoluS095/brand-asset-generator.git
cd brand-asset-generator
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Ideas for Contribution

- New asset engines (video thumbnails, email templates)
- Additional AI model integrations
- Batch processing improvements
- Template library expansion

## Code Style

Follow PEP 8 and use type hints where possible.
